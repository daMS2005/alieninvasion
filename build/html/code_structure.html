Code Structure¶

The Alien Invasion project is organized into the following components:

  • main.cpp: Entry point of the game, handles the main game loop and state transitions.

  • Player.cpp / Player.hpp: Defines the player’s behavior, health system, and projectile interactions.

  • Alien.cpp / Alien.hpp: Implements alien movement, animations, and interactions.

  • Projectile.cpp / Projectile.hpp: Handles projectile behavior and collision detection.

  • GameState.cpp / GameState.hpp: Manages game states like Start Menu, Gameplay, and Game Over.

Resource Directory¶

  • Images: Skins and background assets.

  • Sounds: Background music and sound effects.

  • Fonts: Fonts used for UI elements.
